Detailed Description

template<UInt NNODES>
class Triangle< NNODES >

This class implements a Triangle as an objects composed by three or six nodes.

The first three nodes represent the vertices, the others the internal nodes, following this enumeration: !IMPORTANT! different from Sangalli code!

3 * / \ 5 * * 4 / \ *______*______* 1 6 2

the coordinates usefull for point location are the first three, so I always use 0,1,2 index independently if the class has three or six nodes! is only a 2D triangle, so the correspondig bounding box must have NDIMP = 2
