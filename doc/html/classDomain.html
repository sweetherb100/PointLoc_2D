Detailed Description

template<class Shape>
class Domain< Shape >

It defines geometric limits of a set of points. template parameter Shape is the original geometric figure (ex: triangle, box), it's useful to extract the physical dimension.

Parameters:
[in] coord Point coordinates organized in a matrix (coord[i, j] is the i-th coordinate of the j-th point).
This matrix is created through a vector of vectors in order to use standard algorithms.

It finds geometric limits of a set of points first. Then adds a tolerance. Repeats the limits if the tree dimension is 2 * physical space dimension. This is an useful trick. For example, when you have to scale dimensions.
